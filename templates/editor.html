导出说明
PVE配置文件 (.conf)
  • 直接复制到 /etc/pve/qemu-server/ 目录
  • 使用 qm create 命令创建虚拟机
  • 需要手动创建虚拟磁盘
  • 适用于已有PVE环境
Libvirt配置文件 (.xml)
  • 使用 virsh define [文件] 导入
  • 使用 virsh start [虚拟机名] 启动
  • 需要手动创建虚拟磁盘
  • 适用于KVM/QEMU环境
一键部署脚本 (.sh)
  • 自动创建配置文件和虚拟磁盘
  • 包含错误处理和详细日志
  • 需要root权限执行
  • 支持多种磁盘格式转换
  • 自动备份和验证
